WEBSITE WEDANG DAYAK

Isi folder:
- index.html  : halaman utama website
- styles.css  : desain/tampilan website
- script.js   : interaksi produk, pencarian, modal, dan tombol WhatsApp
- assets/     : logo, foto produk, gambar katalog, dan aset visual

Cara membuka:
1. Extract file ZIP.
2. Klik 2x index.html untuk preview di browser.
3. Untuk upload ke hosting, upload semua isi folder ini ke public_html / htdocs / root website.

Catatan edit cepat:
- Nomor WhatsApp ada di index.html dan script.js, saat ini memakai 6282154143038.
- Daftar produk, harga, dan komposisi ada di script.js pada bagian const products.
- Warna utama ada di styles.css pada bagian :root.
